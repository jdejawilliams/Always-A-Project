Overview

Programming Motion

Programmed an Arduino microcontroller to operate a remote-controlled toy car, implementing wireless signal handling to control movement and steering. This project served as an introduction to embedded systems programming and real-time hardware control.

The key challenge was mapping wireless receiver inputs to reliable motor and steering outputs, managing signal latency and interference at various operating ranges.

Technical Details

Code & Circuit

The firmware was written in C++ using the Arduino IDE, using interrupt-driven signal reading from an RF receiver module. PWM signals were generated to control the DC drive motors and servo-based steering mechanism.

Key lessons included understanding pulse-width timing, debouncing digital inputs, and structuring a control loop that stays responsive at low latency.
